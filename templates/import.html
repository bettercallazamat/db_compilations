Expected JSON Format

Your JSON file should contain a "videos" array with the following structure: 

{
  "videos": [
    {
      "title": "Video Title",
      "video_id": "unique_video_id",
      "published_at": "2024-01-01",
      "description": "Video description",
      "thumbnail_url": "https://example.com/thumbnail.jpg",
      "duration": "5:30",
      "duration_seconds": 330,
      "view_count": 1000,
      "like_count": 50,
      "comment_count": 10,
      "estimated_minutes_watched": 2500,
      "average_view_duration": 150,
      "average_view_percentage": 45.5,
      "retention_30s": 85.2
    }
  ]
}
Required Fields:
  • title - Video title
  • video_id - Unique identifier
Optional Fields:
  • published_at - Publication date
  • description - Video description
  • thumbnail_url - Thumbnail image URL
  • duration_seconds - Duration in seconds
  • view_count - Number of views
  • average_view_percentage - Retention percentage
